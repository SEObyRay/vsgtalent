# VSGTalent WordPress Setup Plugin

WordPress plugin voor automatische configuratie van de VSGTalent backend.

## Installatie

1. Upload `vsgtalent-setup-v1.6.1.zip` naar WordPress
2. Activeer de plugin
3. Ga naar WordPress editor en je ziet het "Media Galerij & Video's" paneel
